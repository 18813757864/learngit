基于 openlayers4 封装的组件。该组件封装了一些常用的微交互单元。但实际开发过程中，可能会出现各种各样的交互，所以要想用好地图，openlayers的官网API也是要会的

基础功能涉及到的指令有 openlayers, olCenter, olControl, olView, olInteraction

openlayers 是整个框架的最基础的指令。 olCenter, olView 都是作为元素openlayers的属性存在的。

olCenter设置地图中心坐标以及缩放等级。olView设置maxZoom，minZoom, rotation。olDefaults可以设置简单的交互动作与控件，这个属性和服务olMapDefaults关系很大。customLayers这个属性若不设置就会加载默认的底图。

若想要监控地图事件，需要在ol-defaults中配置events。并接受消息：

$scope.$on("openlayers.map.singleclick", function (e, data) {}
$scope.$on("openlayers.map.pointermove", function (e, data) {}
